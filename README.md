## GENERAL WORKFLOW FOR EACH PROJECT

Each notebook follows a consistent workflow:

1. **Import Libraries:** Load required Python packages
2. **Load Data:** Read and import the dataset
3. **Data Exploration:** Understand data structure, size, and content
4. **Data Cleaning:** Handle missing values, duplicates, and inconsistencies
5. **Exploratory Data Analysis (EDA):** Visualize data patterns and relationships
6. **Feature Engineering:** Create and transform features for modeling
7. **Data Preprocessing:** Normalize/scale data, encode categorical variables
8. **Model Building:** Implement appropriate ML algorithms
9. **Model Training:** Fit the model on training data
10. **Model Evaluation:** Assess performance using relevant metrics
11. **Visualization:** Display results, predictions, and insights

## HOW TO RUN THE NOTEBOOKS

### Prerequisites
- Python 3.7+
- Jupyter Notebook
- Required libraries (install via pip)

### Installation

```bash
pip install numpy pandas matplotlib seaborn scikit-learn scipy
```

### Running Notebooks

1. Navigate to the project directory
2. Start Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
3. Click on any notebook file to open it
4. Run cells sequentially using Shift+Enter or Run All option
